# Spacing Tokens

**Version:** 1.0.0
**Source:** materials-notes.md §Spacing; raw-observations.md §Spacing — approved by guardian 2026-05-01

All spacing values are multiples of 4px.

| Token Name | Value | Description |
|---|---|---|
| `--spacing-1` | `4px` | Minimum gap, icon margin |
| `--spacing-2` | `8px` | Small padding (sm button v, input h) |
| `--spacing-3` | `12px` | Medium horizontal button padding |
| `--spacing-4` | `16px` | Standard component padding |
| `--spacing-6` | `24px` | Component group spacing |
| `--spacing-8` | `32px` | Section separation |
| `--spacing-12` | `48px` | Major section gaps |
| `--spacing-16` | `64px` | Page-level separation |
