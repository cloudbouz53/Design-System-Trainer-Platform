# Color Tokens

**Version:** 1.0.0
**Source:** materials-notes.md §Color; raw-observations.md §Color — approved by guardian 2026-05-01
**Changelog:** See `../rule-engine/changelog.md` entry: 2026-05-01 — TOK color tokens promoted

| Token Name | Value (light mode) | Description |
|---|---|---|
| `--color-background` | `#ffffff` | Default page and panel background |
| `--color-background-secondary` | `#fafafa` | Card, surface, disabled input background |
| `--color-text-primary` | `#0a0a0a` | Primary body and heading text |
| `--color-text-secondary` | `#666666` | Secondary, muted text, placeholders |
| `--color-border` | `#e6e6e6` | Default borders for inputs, cards |
| `--color-interactive` | `#0070f3` | Links, focus rings, primary CTA background |
| `--color-interactive-ring` | `rgba(0, 112, 243, 0.25)` | Focus shadow ring (transparent interactive) |
| `--color-error` | `#e00000` | Error state borders and text |
| `--color-success` | `#0a7a00` | Success state borders and text |
| `--color-warning` | `#f5a623` | Warning indicators |
