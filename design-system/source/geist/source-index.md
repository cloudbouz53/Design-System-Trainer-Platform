# Geist Source Index

Catalogs what has been inspected from the Geist design system reference.

**Reference:** Vercel's Geist design system (public documentation at vercel.com/geist)
**Research Status:** Initial wave — foundations and Button/Input components

## Sessions

### Session 1 — 2026-05-01: Foundations (materials)

**Coverage:**
- Color palette (backgrounds, text, interactive, semantic colors)
- Typography scale (font family, sizes, weights, line heights)
- Spacing scale
- Border radius values
- Shadow definitions

**Files produced:**
- `raw-observations.md` — verbatim findings with citations
- `materials-notes.md` — structured token tables

### Session 2 — 2026-05-01: Components (Button, Input)

**Coverage:**
- Button: anatomy, variants (primary, secondary, tertiary, ghost), states, accessibility
- Input: anatomy, variants (default, error, success), states, accessibility

**Files produced:**
- `component-notes.md` — per-component structured notes
